# Ecosystem compatibility matrix

| Package | Depends on `python-sendparcel` |
|---|---|
| django-sendparcel | >=0.1.0 |
| fastapi-sendparcel | >=0.1.0 |
| litestar-sendparcel | >=0.1.0 |

## Plugin discovery

The ecosystem entry point group is: `sendparcel.providers`.

The reference `dummy` provider is built into `python-sendparcel`.
